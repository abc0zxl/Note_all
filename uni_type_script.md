# UTS

1.他是一个**编程语言**

* 跨平台
* 高性能
* 强类型

2.**采用和ts一致的语法规范**

3.**他仅仅是一门语言**，不包含ui框架

4.他给变量声明类型的方式比较特殊，用的是**冒号**




## 类型声明

1.**声明变量**：他类型声明要是用的let 和 const

* let可以重新赋值，const则不行
* let可以只声明不赋值，const不行

2.**声明变量类型**：用的是**冒号的方式**

* const str:string="hello";

3.**自动推导类型**：可以通过赋给他的值自动推导出变量类型

* **支持断言方式声明类型**
