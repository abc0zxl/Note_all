# Cloude Code

是一个ai编程工具，他就是一个命令行，它可以操纵电脑文件完成各种任务。

1.**他是一个AI，一个由AI大模型驱动的智能编程助手**

2.**他最大的特点是行动范围大和开放性**

3.**他和通常的豆包，deepseek，chatgpt等的区别**

* **他是一个能在电脑里真实操纵各种文件写入读取的AI代理**


4.**claude目前用到的自己家的anthropic 的claude模型**

5.**他也可以t通过Claude code router截取其他的大模型**

6.**它可以放到各种ai编程工具中使用他**

7.**项目的上线（服务器，域名等）他还不能配置这些东西，**

8.claude适合流程工作，cursor适合调试错误。

# 使用

2.**自定义指令**：/commands
