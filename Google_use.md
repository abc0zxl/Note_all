# 解决google账号在国内容易被封的问题

# 为什么被封

1.**IP地址不干净**：这个ip有非常多的在使用

* **代理的机房ip**：多人在使用，风险值非常高
* **多人共用一个ip**：在那些熟知的翻墙工具中，容易连接多人共用的ip，如果其中一个人有违规操作，会导致这个ip上的所有google连坐被封

2.**频繁换IP**：这个表示用户的google账号短时间内在多个国家切换登录。

* 被认定为**盗窃**，**欺诈**

3.**机器操作**：账号刚注册的时候不正常使用非常容易封

4.**多设备频繁切换**：google采用**环境指纹**识别google使用环境

* 如果短期内在多个非历史记录的设备中来回切换，会判定为**被盗用**。

5.**设备不干净**：设备有过封号记录

* 用**虚拟机，模拟器，被封过号的设备，新裸机**：登录注册
* **有过不良的浏览历史**


# 如何解决？

可以整体分为两个问题

1.**浏览器的不良历史问题**：这个可以通过**指纹浏览器**，伪装成新的账户，抹去历史记录

2.**ip不干净，频繁更换的问题**：这个可以调用干净，危险值小一点的ip。

# IP问题

这个在调用外网的时候，可以控制

他们内部有一个监控，可以查看ip的变动情况，每次变动ip都会记录

1.**ip不动的话**：这个不太现实，非常贵

2.**平时使用的一元机场**：这个用的人太多了，不仅ip不干净，而且人也非常多，违规被封的概率非常大

* 这个不适用于登录google

3.**去贵一点的代理ip网站**：这些ip比较安全，都是住宅ip，安全性大大提高

* DataImpulse:比较便宜，**1$/GB**，用的ip比较干净
* IPRoyal:这个比较贵
* Decode：
* 在AdsPower中也能买得到：
* NovoProxy：这个也比较火

## 判断ip质量的好坏

1.**ASN所有者**：

* ISP：是互联网服务提供商分配给个人用户的家庭IP，纯净度比较高
* IDC：是托管在数据中心的IP，纯净度比较低

2.**双ISP属性**：这个最接近正式家庭住宅网络，安全性最高

3.**家庭，原生IP**：有这个标签，安全性也比较高

4.**ip测试工具**：

* ippure.com可以看到
* ipdata.co查看是否是代理
* ipip.la
