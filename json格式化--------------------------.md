# JSON数据格式化

**问题**：当获取到的数据，在页面上显示时，格式不是期望，易读懂的格式。

**例子**：20261261520，这其实是个时间

**配置类的修改方法**：扩展springmvc的消息转换器

**作用**：统一将后端的数据转换后给前端


## 修改配置信息的JSON处理方法

直接调用固定的代码就好了。**有两个部分**

* **配置处理**：他会调用消息转换器方法
* **json对象转换器**：这个在苍穹外卖的资源文件里面，JacksonObjectMapper

**配置处理步骤**：

1.**方法参数部分**：

* 因为是一个功能，返回是空，
* 参数是一个系统的消息转换器列表对象

2.**代码部分**：

* 创建一个消息转换器对象
* 在设个消息转换器里设置一个**对象转换器**，用来转换java对象为json数据，会导入**JSON对象转换器**
* 导入这个新建的消息转换器，将参数获取到的**消息转换器列表对象**，并将这个转换器级别设置为0（表示最优先使用）
