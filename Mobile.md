# URL Scheme

**作用**：在移动端可以根据各个应用在手机上注册的URL_scheme跳转过来

这个技术适用于移动端Android，IOS。

* **不适用于小程序，因为他们是封闭管理的微信小程序不可能打开支付宝app**
