# 文本显示

1.**标题显示**

**关键字**:h1~h6

**h1**到**h2**有6个级别

2.**正文显示**

**关键字**:p

3.**加粗,倾斜,下划线,删除线,(有语义的)**

**关键字**:<strong<em<ins<del

4.**注释**


# 块级元素,内联元素

就是他有默认的区域,例如标题标签和段落标签就会独占一行

而**加粗,下划线等**:他不会占一整行,就是内联元素

* 内联元素不可以嵌套别的标签

# 图片标签

**关键字**:<img

**属性**

* src:图片的地址
* alt:图片为加载出来的时候,显示的文本


# 音频视频

1.视频显示

**关键字**:<video<source

**属性**

* src:视频地址
* 可以设置大小
* 循环播放
* 自动播放
* 默认静音
* 加载视频时的视频封面

**注意**

* 不同浏览器的支持不一样,通用时mp4

**视频兼容标签**:可以在标签中设置不同格式的视频,浏览器可以自动识别到支持的视频,并播放

* <source<type

![image.png](/assets/c172cf2b-03c3-45e9-85f9-9f8efc90c141.png)

2.音频显示

**关键字:**<audio<source

![image.png](/assets/f9cbf032-b1aa-4008-9c6b-8cb8ff3287ee.png)

**属性**

* 默认静音
* 不可以自动播放

**注意**

* 也会有兼容性问题
