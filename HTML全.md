# 块级元素,内联元素

就是他有默认的区域,例如标题标签和段落标签就会独占一行

而**加粗,下划线等**:他不会占一整行,就是内联元素

* 内联元素不可以嵌套别的标签

# 图片标签

**关键字**:<img

**属性**

* src:图片的地址
* alt:图片为加载出来的时候,显示的文本

# 音频视频

1.视频显示

**关键字**:<video<source

**属性**

* src:视频地址
* 可以设置大小
* 循环播放
* 自动播放
* 默认静音
* 加载视频时的视频封面

**注意**

* 不同浏览器的支持不一样,通用时mp4

**视频兼容标签**:可以在标签中设置不同格式的视频,浏览器可以自动识别到支持的视频,并播放

* <source<type

# 超链接

可以跳转到**外部链接**,也可跳转到**内部链接**

![image.png](/assets/6dea5624-d9c4-4ccb-ab34-d96934b60f4b.png)

**关键字:**<a

**属性**

* href:要跳转的连接,**锚点,网页,邮箱,文件**
* **title**:鼠标悬停的时候给的提示
* ![image.png](/assets/f2c15a20-197d-4c49-8e33-0be846454d91.png)
* **target**:设置是否从当前页面打开

**注意:**

* 在href上设置文件,还可以下载文件
* 可以设置邮箱,可以设置各个运营商的邮箱可以实现到官网的跳转
* **锚点连接**:可以业内的跳转,变动滚动条的位置
* ![image.png](/assets/2d530b7c-c9d0-412a-8712-d40cc7fd5957.png)

**优化**

* **页内跳转实现滑动效果**:在style中添加滑动效果
* ![image.png](/assets/40fabfad-28f1-45f9-8db0-d1e842ae0572.png)
