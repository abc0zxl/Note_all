# HttpClient

他是用于和微信交互的一个工具。

他是apache的一个子项目，可以提供高效，罪行的丰富的支持，http协议的一个客户端编程工具包。

**功能**：通过java变成的方式来，构造http请求，发送http请求

**依赖**：

**核心API**：

1.HttpPost：post请求

2.HttpGet：get请求

3.**CloseableHttpClient**：一个具体实现类，实现了HttpClient这个接口

4.**HttpClients**：构建器，用于创建httpclient对象

5.**HttpClient**：发送http请求
