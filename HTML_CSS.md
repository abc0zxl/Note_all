# 文本样式

### **字体样式**:font

* color
* font-family:字体
  1.**微软雅黑**
* font-size:字体大小
  1.**px**
* font-style:字体风格
  1.**倾斜字体**
* font-weight:字体粗体
  1.**通过数值来这只加粗程度,默认的是400**
* text-decoration:字体装饰:**下划线,删除线等等**
  1.**none**:没有线
  2.**underline**
  3.**overline**
  4.**line-through**

### **文本布局**:

控制文本在**盒子中的位置**

1.**text-align**:控制文本在盒子内如何水平对齐

* left
* right
* center
* justify自动改变字间距,两端对齐

2.**text-indent**:指定缩进

* **单位**:em,px,lem
* **em**:以字的大小为单位

3.**letter-spacing**:字体间距

* 单位:px

4.**line-height**:设置行的高度

* 单位:**纯数字**,px
* **纯数字单位**:表示的是以当前一个字为单位长度
* **px**:通常单位
